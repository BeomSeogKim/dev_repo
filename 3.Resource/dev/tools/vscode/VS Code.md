## Shortcut
- Extension : command + shift + x
- 폴더 / 파일구조 보기 : command + shift + e
- 명령 팔레트 : command + shift + e
- vs code 설정 : command + ,
- 왼쪽 창 숨기기 보이기 : command + b
- 터미널 닫기 열기 : command + j
- 자동 정렬 : shift + alt + f
